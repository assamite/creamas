Rule¶

Rule module holds base implementation of rule (Rule). Rules combine features and mappers to functional body, where each feature also has weight attached to it.

class creamas.core.rule.Rule(feats, weights)[source]¶

Rule has a collection of features (F), a mapping of features to the values in interval [-1, 1] (mappers) and weight for each feature (W).

Like features, rules can be directly evaluated for artifacts by calling them with artifact as the only argument.

from myrules import MyRule
feats = [myfeat1, myfeat2]
weights = [0.5, 0.3]
types = [mytype]
mr = MyRule(feats, weights, types)
mr.mappers([feat1_mapper, feat2_mapper])
rule_result = mr(myartifact)
evaluate(artifact)[source]¶

Evaluate artifact with this rule. Called when the instantiated object is called.

F¶

list - features in this rule.

W¶

list - weights for features in this rule.

domains¶

Rule’s acceptable artifact domains is the union of all its features acceptable domains. Each artifact is evaluated only with features that do not return None when the feature is extracted from it.

mappers¶

list - mappers for features in this rule.
