Agent ConnectionsÂ¶

Creamas offers a capability to create agent connections from NetworkX graph structures and create NetworkX graphs from existing agent connections. These functionalities are found from creamas.nx module.

The main functions in creamas.nx are connections_from_graph() and graph_from_connections().
