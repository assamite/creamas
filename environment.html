Environments for agents.

class environment.Environment(addr, clock=None, extra_serializers=None, loggers=[])[source]¶

Basic environment for agents.

Environment contains aiomas.Container as a communication route between agents and has artifacts variable that holds all the artifacts agents have published.

add_artifact(agent, artifact, framing)[source]¶

Add artifact with give framing to environment.

create_initial_connections(n=5, type='random')[source]¶

Create random initial connections for all agents.

destroy(folder=None)[source]¶

Destroy the environment.

Destroying the environment calls all its agents shutdown method, and then shuts down the agent container. If folder is given, saves information of the run to that folder.
