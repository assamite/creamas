Util¶

Various utility functions for logging, etc.

creamas.util.log_before(attr, level=20, sep='\t')[source]¶

Decorator to log attribute’s value(s) before function call.

Implementation allows usage only for methods belonging to class. The class instance needs to have logger attribute that is subclass of creamas.core.logger.ObjectLogger.

Parameters:
  • level (int) – logging level
  • attr (str) – name of the class instance’s parameter to be logged
  • sep (str) – if attr is iterable, separate its elements with sep
